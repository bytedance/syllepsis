`Syllepsis` supports dynamic copywriting configuration to complete international copywriting

## Use

### Local parameters

Support the input of `locale` parameter, locale is the corresponding multi-language text

```jsx
const locale = {
bold: {
tooltip:'Bold'
}
}

// ...
<SylEditor
    locale={locale}
/>
```



# Configuration
