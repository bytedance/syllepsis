# 安装 <!-- {docsify-ignore-all} -->

## 下载

```shell
# npm
npm install @syllepsis/access-react

#yarn
# yarn add @syllepsis/access-react
```

## 使用

```jsx
import { SylEditor } from '@syllepsis/access-react';

export default function App() {
  return <SylEditor />;
}
```
